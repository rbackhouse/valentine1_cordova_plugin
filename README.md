Valentine1 Cordova Plugin
=========================

Cordova Plugin for the Valentine1 Radar Detector

Installation
============

1) Download the Valentine1 ESP Development Library for the Apple iOS from http://www.valentine1.com/eula/eula.asp?d=espdevLibiOS
2) Unzip and copy the ESPLibrary directory to valentine1_cordova_plugin/src/ios directory
3) Run

cordova plugin add <path to>/valentine1_cordova_plugin
